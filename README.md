## Getting Started

### Installation

1. Clone the repo

   ```sh

   git clone https://github.com/aiseatu/color-mixing.git

   ```

2. Install NPM packages

   ```sh

   npm install

   ```

3. Run on local

   ```sh

   npm run start

   ```
